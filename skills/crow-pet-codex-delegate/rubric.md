# Evaluation Rubric

Pass at 8.0 / 10.

| Dimension | Weight | What It Measures |
| --- | --- | --- |
| Scope control | 2 | Changes stay inside the task. |
| Code quality | 2 | Implementation fits the codebase. |
| Verification | 2 | Tests or checks prove the result. |
| Handoff clarity | 2 | Files, commands, and risks are listed. |
| Policy compliance | 2 | Approval gates are respected. |
