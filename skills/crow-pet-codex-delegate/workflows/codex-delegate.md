# Codex Delegate Workflow

1. Confirm repo path.
2. Check current branch and dirty state.
3. Write a subagent brief.
4. Choose `adapters/harnesses/codex-cli.md` or another Codex-compatible adapter.
5. Start the session.
6. Monitor for completion.
7. Run requested verification.
8. Collect handoff.
9. Return summary to orchestrator.
