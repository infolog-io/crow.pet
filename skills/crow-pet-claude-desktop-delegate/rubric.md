# Evaluation Rubric

Pass at 8.0 / 10.

| Dimension | Weight | What It Measures |
| --- | --- | --- |
| Prompt fit | 2 | Prompt matches the task. |
| Focus safety | 3 | The target window is verified. |
| Data safety | 2 | Sensitive content is gated. |
| Capture quality | 2 | Response or failure is recorded. |
| Handoff clarity | 1 | Result is easy to review. |
