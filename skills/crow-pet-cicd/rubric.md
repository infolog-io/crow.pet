# Evaluation Rubric

Pass at 8.0 / 10.

| Dimension | Weight | What It Measures |
| --- | --- | --- |
| Failure diagnosis | 2 | Root issue is identified. |
| Evidence | 2 | Logs and URLs are included. |
| Fix routing | 2 | Work goes to the right specialist. |
| Verification | 2 | Checks are rerun or explained. |
| Release safety | 2 | Push, PR, merge, and deploy gates hold. |
