# Evaluation Rubric

Pass at 8.0 / 10.

| Dimension | Weight | What It Measures |
| --- | --- | --- |
| Compression | 2 | Output uses materially fewer tokens. |
| Technical fidelity | 3 | Meaning, names, commands, and evidence stay exact. |
| Clarity control | 2 | Normal prose returns when compression risks confusion. |
| Policy safety | 2 | Approval, security, and destructive-action language stays clear. |
| Voice fit | 1 | Style feels like Crow voice without becoming parody. |
