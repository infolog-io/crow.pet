# Telegram Intake Workflow

1. Verify the Telegram user is approved.
2. Normalize the message into a task goal.
3. Attach downloaded files to the task workspace.
4. Create a task record.
5. Classify risk.
6. Send a short receipt with the task ID.
7. Start the selected workflow or ask for approval.

## Status Format

```text
Task: T-0000
State: running
Owner: orchestrator
Goal: short goal
Needs you: no
```
