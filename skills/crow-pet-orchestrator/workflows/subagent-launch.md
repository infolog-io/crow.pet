# Subagent Launch Workflow

1. Choose one primary specialist.
2. Choose the required adapters.
3. Write a subagent brief.
4. Include success criteria.
5. Include policy constraints.
6. Include verification steps.
7. Start the subagent.
8. Monitor for stall or failure.
9. Collect the handoff report.
10. Run evaluation.

Retry once only when policy allows.
