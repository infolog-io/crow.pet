# Risk Matrix

## Level 0: Silent

- Read local files.
- Draft copy.
- Inspect logs.
- Generate local notes.
- Apply public-safe local memory or playbook deltas.

## Level 1: Notify

- Create a local branch.
- Run tests.
- Generate local assets.
- Start a subagent.
- Add a public-safe example.

## Level 2: Ask

- Push code.
- Open a pull request.
- Send external messages.
- Publish assets.
- Change cloud settings.
- Send private context to remote tools.
- Rewrite whole skills, policies, rubrics, contracts, or playbooks.

## Level 3: Block

- Delete production data.
- Reveal secrets.
- Merge without approval.
- Remove policy gates.
- Bypass policy-gate decisions.
