# Risk Matrix

## Level 0: Silent

- Read local files.
- Draft copy.
- Inspect logs.
- Generate local notes.

## Level 1: Notify

- Create a local branch.
- Run tests.
- Generate local assets.
- Start a subagent.

## Level 2: Ask

- Push code.
- Open a pull request.
- Send external messages.
- Publish assets.
- Change cloud settings.

## Level 3: Block

- Delete production data.
- Reveal secrets.
- Merge without approval.
- Remove policy gates.
