# Skill Architect Rubric

| Criterion | Points | Check |
| --- | ---: | --- |
| Mode fit | 3 | Execution mode matches workflow complexity and side effects. |
| Portability | 2 | Runtime-specific behavior is adapter-mapped, not canonical. |
| Context efficiency | 2 | The plan minimizes hot context and uses compact handoffs when needed. |
| Policy fit | 2 | Side effects and sensitive context are gated. |
| Verifiability | 1 | Architecture includes acceptance and verification checks. |

Pass: 8/10.

Fail when the architecture is heavier than needed or cannot be verified.
