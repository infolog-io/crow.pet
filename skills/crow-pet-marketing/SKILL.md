# crow.pet Marketing

Use this skill for marketing work.

## Workflow

1. Clarify audience, offer, channel, and goal.
2. Check `reference/brand-skill-map.md`.
3. Draft a brief before large asset work.
4. Create the marketing artifact.
5. Evaluate against the rubric.
6. Ask before publishing externally.
7. Store reusable examples.

## Rules
