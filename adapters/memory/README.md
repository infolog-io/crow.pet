# Memory Adapters

Memory adapters keep markdown useful without making it huge.

They decide what context enters a task.

They do not own the canonical skill tree.

## Layers

- Hot memory: tiny facts always safe to inject.
- Warm memory: task-specific capsules selected on demand.
- Cold memory: raw logs, wiki pages, session archives, and providers.

## Rule

Markdown files stay navigational.

Large records stay in raw sources, archives, provider stores, or external systems.

Every selected excerpt needs a source and a reason.

Use context capsules for task-specific recall.

Never paste full `USER.md`, `SOUL.md`, or wiki folders by default.
