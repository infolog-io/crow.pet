# Target Adapters

Target adapters describe how to export canonical crow.pet skills.

The source of truth stays in `../../skills`.

The shared project guide is `../../AGENTS.md`.

## Targets

| Target | Native shape |
| --- | --- |
| ChatGPT | Custom GPTs, Projects, GPT Actions. |
| Codex | Agent Skills, AGENTS.md, workflows. |
| Cursor | Skills, rules, memories, modes, subagents. |
| Copilot VS Code | Instructions, prompts, agents, skills. |
| Gemini CLI | GEMINI.md, memory, custom commands. |
| Windsurf | Rules, Workflows, Skills, Memories. |
| Cline | Rules, Skills, Workflows, Hooks. |
| OpenCode | Skills, AGENTS.md, custom agents. |
| Devin | Skills, Playbooks, Knowledge, subagents. |
| Aider | Convention files, repo map, AGENTS.md. |
| Hermes | Skills, context files, bounded memory, providers. |
| OpenClaude | Claude Code config, Hindsight, workspace files. |
| Obsidian Skills | Obsidian markdown, Bases, Canvas, CLI, Defuddle. |

## Rule

Regenerate target files from canonical skills.

Do not edit generated target output by hand.
